Biodiversity Visualization in State Parks


Description: This custom visualization depicts the biodiversity of state parks across the US. Each park is represented by a pine tree marker on a US map, with the size of the pine tree correlating to the area of the park.

Marks:

  1. Pine tree markers represent each state park on the US map.
  2. Donut chart segments represent different attributes (order, family, category) of organisms within a selected park.

Channels:

  1. Position on the map indicates the geographic location of each state park.
  2. Size of the pine tree markers encodes the area of each park.
  3. Color of the donut chart segments can be used to distinguish different categorical values of order, family and category attributes
  4. Text: Displays the acronym of the state name inside the boundary of each state, providing additional information to users.

Interactions:

  1. Hovering over a state on the map displays the full name of the state, providing users with additional context and information about the location they are interacting with.
  2. On hovering over a park, it enlarges to indicate selection and displays the park's name.
  3. Clicking on a park reveals a donut chart illustrating the distribution of organisms by order, family, and category within that park.
  4. The donut chart is interactive, allowing users to hover over segments to view percentages and names of attributes.
  5. Clicking on segments of donut charts displays all organisms belonging to the selected attribute within the park.

Category: Combinatorial
Explanation: This visualization falls into the combinatorial category because it combines multiple established data visualization techniques (geographical mapping, symbol size encoding, interactive hover and click functionality, donut chart representation) to effectively convey complex biodiversity information. It integrates these techniques seamlessly to provide users with an interactive and informative experience, making it a clear use case for custom visualization.

Attributes visualized:
  1. Longitude and Latitude
  2. State
  3. Acres
  4. Park Name
  5. Category
  6. Order
  7. Family
  8. Common Names
